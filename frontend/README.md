# React Health Status Dashboard

This React application displays the health status of the JavaSpringBoot TODO backend API.

## Features

- Fetches backend health status from `http://localhost:8080/api/health`
- Uses `useState` and `useEffect` React hooks
- Auto-refreshes every 5 seconds
- Displays error message when backend is down
- Manual refresh button
- Beautiful, responsive UI

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- JavaSpringBoot TODO backend running on port 8080

## Installation

```bash
cd frontend
npm install
```

## Running the Application

The React app is configured to run on port 3000:

```bash
npm start
```

The application will automatically open in your browser at `http://localhost:5173`

## Testing Backend Down Scenario

1. Start the React app (it will show an error if backend is not running)
2. Start the JavaSpringBootDemo backend on port 8080
3. The React app will automatically detect the backend is up
4. Stop the backend to see the error message appear
